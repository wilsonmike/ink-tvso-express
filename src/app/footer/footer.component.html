View Different Month

The green "Change Month" button on the homepage allows you to switch views between months. A selected month's landing page includes the total cost of all stores for that month with the totals for each individual store below (underlined in green). If you want to view more order details there is a button for each store which routes to that stores data table.

View By Store

"View By Store" is located in the upper right-hand corner of the navigation bar. After clicking the dropdown menu you will see a list of all stores that had orders fulfilled by InkPressions that month. This is where you can click a store for more order details and data behind the overall totals.
